# @vault/ui

Shared UI components for Vault applications.

## Usage

```typescript
import { Button } from '@vault/ui'

function App() {
  return (
    <Button variant="primary" size="md">
      Click me
    </Button>
  )
}
```

## Components

- **Button**: A customizable button component with variants and sizes

## Development

This package uses TypeScript and React. Components should be generic and reusable across both desktop and web apps.
